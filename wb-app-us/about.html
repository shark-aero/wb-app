Shark.aero weight and balance is a web application that helps to calculate CG position and weight of the aircraft. The app is accessible at app.shark.aero. It runs with a web browser on mobile and computer. After the first visit, the app is usable offline.

How to use

  • Enter the empty weight and CG position written in your flight manual.
  • Use the sliders or the input boxes to enter weights loaded.
  • The ballast position is set automatically depending on the rear seat load.
  • The weight and balance result is drawn on the figure bellow the inputs. The x-axis of the graph corresponds to the CG position, given in % of mac. The y-axis corresponds to the weight.

Both maximal take-off weight and zero-fuel weight configurations are drawn. If one of these configurations is out of the CG envelope, a warning appears on the graph.

Caution messages will appear if the occupants weight is greater than 200kg, if the fuel quantity is greater than 100L with 2 occupants or the baggage weight greater than 15kg with two occupants.

How to install on mobiles

On mobiles, it is possible to install the app like a native app.

  • Chrome browser automatically displays a banner that propose to install the app on the home screen. If the banner does not appear, tap the menu button at the top right corner and select “Install the application”.
  • On safari, it is needed to tap the “share button” then select “Add to home screen”.
  • On Firefox, tap the menu button at the top or bottom right corner and select “Add to home screen”.

The app takes care of the updates by itself. However, it is still possible to force the update by deleting the cache associated to the application in the browser setting. From the next visit of the website, the last version online will be loaded.

Last inputs entered are saved by the browser/app. Carefully check that empty weight and CG correspond to the aircraft you intend to use.
